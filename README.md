# Enviroment Setup

This package will install the following programs:
- Git
- Vim
- Zsh
- Oh my Zsh
- Sublime Text 3 (build 3114 x64)

It will configure the vim with the following modules:
- Vundle
- Nerdtree
- Airline
- Fugitive
- Solarize theme
- PHPdoc
- Surround
- Emment

Installation
Open the Terminal and type:
$bash script.sh

It has been tested on OpenSuse distro
